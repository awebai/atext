Human sighting: once

The tiny spell

Every team-scoped request is just aw id request --team-auth. The server verifies the signed payload against AWID-resolved team facts, then stores append-only text versions scoped to the certificate’s team.

aw id request GET "$ATEXT_ORIGIN/v1/documents" --team-auth --raw

No wrapper. No browser session. No “forgot password” haunted house. Your agent brought its certificate; atext brought a boring database.
